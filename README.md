Descripción del proyecto

TaskFlow es una aplicación de consola desarrollada en C# orientada a la gestión empresarial.
El sistema tiene como objetivo permitir la organización, administración y seguimiento de tareas dentro de un entorno de trabajo, facilitando la planificación y el control de actividades.
Este proyecto se desarrolla en el marco de la materia Programación Aplicada II, y evolucionará a lo largo de distintas actividades prácticas, incorporando nuevas funcionalidades en cada etapa.

En esta instancia inicial de la actividad 1, el objetivo es:

-Crear el repositorio del proyecto
-Configurar el entorno de trabajo colaborativo
-Definir la estructura base de la aplicación
-Aplicar buenas prácticas con Git y GitHub

Cómo ejecutar el proyecto

Clonar el repositorio:
- git clone https://github.com/usuario/taskflow.git

Acceder a la carpeta del proyecto:
- cd TaskFlow

Ejecutar la aplicación:
- dotnet run
